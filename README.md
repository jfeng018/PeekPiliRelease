## ⚠️ 注意事项

- **Windows 端安装**：安装时请避免使用中文路径，建议使用英文路径，以免出现兼容性问题
- **视频源配置**：建议优先使用在线链接配置视频源，本地包在部分设备上可能存在识别问题，iOS 设备最为明显
- **JAR包支持**：安卓端正式支持JAR包运行，配置方式与影视源相同
- **小说功能**：需要在接口配置中添加"小说"类型源才会显示小说Tab
- **漫画功能**：需要在接口配置中添加"漫画"类型源或名称包含"[画]"才会显示漫画Tab（v1.0.9新增）
- **PHP运行时**：内置PHP环境，本地即可运行T4源，无需搭建服务器（v1.0.8新增）
- **磁力播放**：安卓端支持磁力链接直接播放（v1.0.9新增）
- **道长项目使用**：
  - 使用drpy2时，如遇到部分网页源兼容性问题，请及时反馈
  - 启动了drpys后，如需切换另一个node接口，必须先退出应用并清理后台，再进行切换
  - iOS暂不支持道长项目相关功能
- **iOS用户特别注意**：如需切换nodejs接口，请务必按照v1.0.1中的步骤操作，避免应用闪退
- **TMDB功能**：需要自己申请TMDB API Key才能使用媒体库功能
- **Win端优化**：大概率修复了win端node进程残留问题

## 项目简介

**PeekPili** 是一款集成 **TVbox 视频源播放功能**、**小说阅读功能**和**漫画阅读功能**的全能应用，支持多种视频源格式，让您可以在一个应用中享受丰富的第三方视频源、小说和漫画内容。

**✨ 多平台支持** - PeekPili 支持 Android、iOS、Pad、Windows 等平台，无论您使用什么设备，都能享受一致的使用体验。

- **🐧 企鹅群：756526268**

## 适配平台

- [x] **Android** - 手机、平板完美支持
- [x] **iOS** - iPhone、iPad 原生体验
- [x] **Pad** - 平板设备优化适配
- [x] **Windows** - 桌面端完整功能
<br/>

## 技术栈

- **Flutter** - 基于 Flutter 框架开发
- **播放器** - MPV、MDK 和 ExoPlayer（安卓端）

## 核心功能

### 📖 漫画阅读模块（v1.0.9新增）

**PeekPili** 现已支持专业的漫画阅读功能，为漫画爱好者提供沉浸式的阅读体验。

#### 🎯 智能显示机制
- **动态Tab显示**：当接口配置中存在"漫画"类型源时，自动在底部导航栏显示漫画Tab
- **多种识别方式**：支持 `"类型": "漫画"`、名称包含"[画]"等识别方式
- **多源支持**：可在不同漫画源间自由切换，享受丰富的漫画资源

#### 📚 强大的阅读器功能
- **多种阅读模式**：纵向滚动、横向翻页、条漫模式，适应不同类型的漫画
- **图片缩放**：支持图片放大缩小，查看细节更清晰
- **触屏操作**：单击显示/隐藏控制栏，上下滑动滚动阅读
- **PC端支持**：完整的键盘快捷键支持，方向键、PageUp/PageDown等
- **图片格式**：支持 `pics://` 协议，自动解析多张图片链接

#### 🎨 完善的界面设计
- **漫画首页**：分类浏览、筛选功能、网格布局展示
- **漫画详情页**：封面展示、简介、章节列表
- **阅读器界面**：流畅的翻页体验，支持多种阅读模式切换

> 📖 **详细说明**  
> 漫画模块支持 `pics://图片1URL&&图片2URL&&图片3URL...` 格式，系统会自动解析并按 `&&` 分隔提取图片列表。

#### 支持的视频源类型

本项目全平台支持以下三种类型的视频源：

1. **T4 源（TVbox Type4）**
   - 支持标准 T4 API 接口源
   - 支持 HTTP API 格式的视频源
   - **PHP运行时支持** - 内置PHP环境，本地即可运行T4源（v1.0.8新增）
   - 完整的视频分类、搜索、详情、播放功能
   - 支持多站源切换和管理
   - 支持解析接口配置

2. **T3 源 - CatJS 源（猫源）**
   - 支持 cat 格式的 JavaScript 爬虫源
   - 支持 ES Module 格式的 JS 源
   - 支持 **Open 版本**和 **NodeJS 版本**两种猫源格式
   - 内置常用工具库（CryptoJS、lodash、cheerio 等）
   - 支持远程 URL 和本地代码两种方式

3. **T3 源 - Python 源（py 源）**
   - 支持 Python 爬虫脚本源
   - 完整的爬虫接口支持

#### 源助手功能

为了方便用户测试和调试视频源，项目内置了强大的**源助手**工具：

- **CatJS 源调试助手**
  - 支持测试 cat 格式的 JS 爬虫源
  - 支持远程 URL 和代码输入两种模式
  - 可测试所有爬虫方法：
    - `homeContent` - 首页分类
    - `homeVideoContent` - 首页推荐视频
    - `categoryContent` - 分类内容
    - `searchContent` - 搜索内容
    - `detailContent` - 详情内容
    - `playerContent` - 播放内容
  - 实时查看测试结果，支持 JSON 格式化显示
  - 支持清除缓存，方便重新测试

- **Python 源测试助手**
  - 支持测试 Python 爬虫源
  - 支持 URL 和代码两种输入方式
  - 完整的错误提示和调试信息

## 使用说明

### 漫画功能使用（v1.0.9新增）

1. **配置漫画源**
   - 在接口配置中添加 `"类型": "漫画"` 字段或名称包含"[画]"
   ```json
   {
     "sites": [
       {
         "key": "comic_source",
         "name": "漫画源[画]",
         "api": "https://example.com/api/comic",
         "type": 3,
         "类型": "漫画"
       }
     ]
   }
   ```

2. **开始阅读**
   - 配置完成后，漫画Tab会自动出现在底部导航栏
   - 点击漫画Tab，选择漫画源，即可开始阅读

3. **阅读器使用**
   - 单击屏幕中央显示/隐藏控制栏
   - 上下滑动滚动阅读
   - 控制栏 +/- 按钮调整图片缩放
   - PC端支持键盘快捷键（方向键、PageUp/PageDown、Ctrl+滚轮等）

4. **阅读模式切换**
   - 点击控制栏的"模式"按钮切换阅读模式
   - 纵向滚动：适合条漫、长条漫画
   - 横向翻页：适合传统日漫、页漫
   - 条漫模式：连续滚动条漫

> 📖 **图片格式说明**  
> 漫画源返回的图片格式应为：`pics://图片1URL&&图片2URL&&图片3URL...`  
> 系统会自动解析 `pics://` 协议，按 `&&` 分隔提取图片列表。

### 小说功能使用（v1.0.6新增）

3. **阅读器使用**

   - PC端支持键盘快捷键（方向键、PageUp/PageDown、空格键等）

   - 支持搜索建议和首页长按跳转（v1.0.7新增）
   - 选择播放线路和画质
   - **磁力播放**（v1.0.9新增）：支持磁力链接直接播放
   - **安卓端ExoPlayer增强**（v1.0.9新增）：享受杜比/DTS直通、HDR完美显示、多音轨切换等专业播放体验
   - 享受流畅的播放体验
   - **TMDB媒体库体验**（v1.0.4优化）

## 常见问题

### Q: 有没有 TV 端？

**A:** 没有。本项目目前不支持 TV 端。

> 💡 **推荐**  
> 如果您仅仅是安卓手机和 TV 用户，强烈推荐使用 **ok影视**，因为兼容性最强、覆盖性最广。

### Q: 支不支持 jar 包？

**A:** 安卓端正式支持JAR包运行（v1.0.6新增）。配置方式与影视源完全相同，基本适配所有JAR方法，支持GO代理播放功能。目前已测试VOX和摸鱼JAR包。

### Q: 播放是否支持本地代理（如 127.0.0.1:9978 格式）？

**A:** Python源和JAR包（v1.0.6新增）支持。JAR包支持GO代理播放功能。

### Q: 是否支持磁力链接播放？

**A:** 支持（v1.0.9新增）。安卓端已支持磁力链接直接播放功能。

### Q: 安卓端播放器有什么增强功能？

**A:** 安卓端原生ExoPlayer已升级（v1.0.9新增），支持：
- 杜比/DTS音频直通解码
- HDR10/杜比视界完美显示
- 多音轨/字幕自由切换
- 后台播放（锁屏继续播放）
- 实时网速显示

### Q: 如何本地运行T4源？

**A:** 在实验室中启用PHP运行时环境（v1.0.8新增），即可本地运行T4源，无需搭建服务器。详细使用指南请查看：[PHP运行环境使用指南](PHP运行环境使用指南.md)。

### Q: 漫画功能如何使用？

**A:** 需要在接口配置中添加 `"类型": "漫画"` 字段或名称包含"[画]"，配置完成后漫画Tab会自动显示。支持 `pics://` 协议格式的图片链接。

### Q: 小说功能如何使用？

**A:** 需要在接口配置中添加 `"类型": "小说"` 字段，配置完成后小说Tab会自动显示。支持多种识别方式，包括名称包含"小说"、key包含"novel"/"book"等。详细使用说明请查看[小说模块功能文档](README_NOVEL.md)。

### Q: 小说阅读器支持哪些功能？

**A:** 支持个性化设置（字体、主题、翻页模式等）、TTS语音朗读、自动翻页、书签、笔记高亮、屏幕常亮等功能。PC端还支持完整的键盘快捷键操作。
